## Test Framework

Test Framework for testing apps, ability to test web-based JS apps.


### Technology

- JavaScript
- NodeJs

### Deployment

1. Run `npm install` or `yarn`
2. Run `npm start`
3. Change permission to index.js - `chmod +x index.js` (Mac OS or Linux)
4. Link project - `npm link`


### Manual

* To run, use command - `tme`
